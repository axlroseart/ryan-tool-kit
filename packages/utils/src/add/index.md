# add

一个处理两数相加的函数
## 代码演示

### 使用一

```ts
import { add } from '@ryan-drx/utils';

add(1, 2); // 3
```

### 使用二

```ts
import { add } from '@ryan-drx/utils';

add(1, 2); // 3
```
## API

```typescript
const add: (a: number, b: number) => number
```

### Params

| 参数      | 说明                                     | 类型    | 默认值 |
|-----------|------------------------------------------|---------|:-------|
| a | 相加第一个数 | `number` | 无 |
| b | 相加第二个数 | `number` | 无 |
